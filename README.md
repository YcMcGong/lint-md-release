# LintMD

> A calm, fast Markdown reader for macOS — built for the AI-generated documentation era.

---

## The Problem

AI tools generate enormous amounts of Markdown every day — READMEs, architecture specs, sprint plans, technical notes, generated docs. Most developers end up reading these in:

- **VSCode** — a full IDE opened just to read a file
- **GitHub in the browser** — requires a round-trip through git
- **Raw terminal output** — no formatting, hard to scan

These tools weren't designed for reading. They're heavy, distracting, and slow to open.

LintMD is.

---

## What LintMD Does

LintMD is a native macOS app with one job: open a Markdown file or a folder and read it beautifully. Nothing else.

- **Open a file** — drag and drop, ⌘O, or File → Open
- **Open a folder** — browse the entire file tree in the sidebar, auto-opens `README.md`
- **Rendered preview** — headings, code blocks, tables, images, blockquotes, checklists
- **Syntax highlighting** — 40+ languages in plain-text files (Swift, Python, JSON, YAML, and more)
- **⌘F find** — search within any file with match count and prev/next navigation
- **Sidebar filters** — show All files or Markdown Only
- **Edit mode** — make quick edits without switching apps
- **Live file watching** — reloads automatically when an AI agent or external tool writes to the file
- **Open Recent** — jump back to files and folders you opened before
- **Dark and light mode** — follows macOS system appearance

## Who It's For

**AI-heavy developers** — if you use Claude Code, Cursor, Codex, or any AI agent that produces Markdown output, LintMD is the fastest way to read what they write.

**Developers** — browsing local docs, READMEs, and wikis without opening a full IDE.

**Anyone** who wants to read Markdown the way a book is meant to be read — cleanly, quickly, locally.

---

## What LintMD Is Not

LintMD is deliberately simple. It is **not**:

- A note-taking or PKM app
- A cloud sync or collaboration tool
- A writing platform
- An Electron app
- Anything with a database, vault, or plugin system

The simplicity is the product.
